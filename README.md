# Project Name: Web Dice Roller
Project created by Karen Villasenor

This project was made with the help of Google AI Overview and ChatGPT. All parts were tested to make sure they work and to understand what each part does for this project. For example, the table created, the rollDice() function, and randomDie() function.

# Compiling and Executing
To run the project, download or clone the repository, make sure all of the files are in the same folder, and then open the “index.html” file in a web browser
